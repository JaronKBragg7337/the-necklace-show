# Canonical archive

This directory preserves the exact source material and generated project that existed before the ongoing production work in the repository root.

Its purpose is provenance. Visitors should be able to distinguish what Kimi AI K3 produced from one prompt from what was later revised, directed, modeled, recorded, edited, or engineered by people and other tools.

[`kimi-k3-one-prompt-original/`](./kimi-k3-one-prompt-original/) contains:

- [`The_Necklace_Three.js_Book_Show.original.zip`](./kimi-k3-one-prompt-original/The_Necklace_Three.js_Book_Show.original.zip): the untouched ZIP returned from the one-prompt generation.

## What is not part of the one-prompt generation

Files in the repository root represent the continuing production version. Character consistency work, camera cuts, enhanced water and weather rendering, memory-sequence treatments, narration support, documentation, future 3D assets, recorded performances, and later revisions must not be described as part of Kimi's original one-prompt output.

The archive should remain unchanged. If additional original source material is discovered, add it with a provenance note and checksum rather than altering the existing files.
